# Sales_prediction_analysis
Project Overview
This project predicts product sales based on advertising spend across different marketing platforms such as TV, Radio, and Digital Media.
The objective is to analyze how advertising investment impacts sales and to generate actionable business insights using Machine Learning.

Objectives
Perform data cleaning and preprocessing
Conduct Exploratory Data Analysis (EDA)
Build a regression model to predict sales
Evaluate model performance
Analyze the impact of advertising channels on sales

Tech Stack
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Google Colab 

Workflow
1 Data Collection
2 Data Cleaning & Handling Missing Values
3 Encoding Categorical Variables
4 Exploratory Data Analysis
5 Train-Test Split
6 Model Building (Linear Regression)
7 Model Evaluation (R² Score, MAE)
8 Business Insights
