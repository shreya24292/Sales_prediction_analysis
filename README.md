Sales Prediction Analysis
Project Overview

This project focuses on predicting product sales based on advertising expenditure across multiple marketing channels, including TV, Radio, and Digital Media. By leveraging Machine Learning techniques, the analysis identifies the relationship between advertising investments and sales performance, enabling data-driven marketing decisions and budget optimization.

🎯 Objectives
Clean and preprocess advertising and sales data.
Perform Exploratory Data Analysis (EDA) to uncover patterns and trends.
Build a Linear Regression model to predict sales.
Evaluate model performance using standard regression metrics.
Analyze the impact of different advertising channels on sales outcomes.
Generate actionable business insights for marketing strategy optimization.

🛠️ Tech Stack
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Google Colab

📂 Project Workflow

1. Data Collection
Import and inspect the advertising dataset.
Understand data structure and feature distribution.

2. Data Preprocessing
Handle missing values and duplicates.
Perform data cleaning and validation.
Prepare data for analysis and modeling.

3. Exploratory Data Analysis (EDA)
Analyze relationships between advertising channels and sales.
Visualize trends, correlations, and feature distributions.
Identify key factors influencing sales performance.

4. Feature Engineering & Encoding
Transform and encode variables where necessary.
Prepare features for machine learning algorithms.

5. Train-Test Split
Split the dataset into training and testing sets to ensure unbiased model evaluation.

6. Model Development
Build a Linear Regression model to predict sales based on advertising spend.

7. Model Evaluation

Evaluate model performance using:

R² Score (Coefficient of Determination)
Mean Absolute Error (MAE)
Additional regression metrics where applicable

8. Business Insights
Determine which advertising channels contribute most to sales growth.
Provide recommendations for effective marketing budget allocation.
Highlight opportunities for improving return on advertising investment (ROAS).

📈 Expected Outcome

The final model provides accurate sales predictions and delivers valuable insights into the effectiveness of various advertising channels, helping businesses make informed marketing and investment decisions.
